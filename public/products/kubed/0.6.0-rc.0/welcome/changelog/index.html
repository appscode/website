0.1.0 (2017-08-01)

First public release of Kubed.

Notable Features

  • Takes periodic snapshot of a Kubernetes cluster objects.
  • Provides a recycle bin for deleted and/or updated Kubernetes objects.
  • Keep ConfigMaps and Secrets synchronized across Namespaces.
  • Forward cluster events to various destinations.
  • Setup janitors for Elasticsearch and InfluxDB.
  • Send notifications via Email, SMS or Chat.
  • Includes a built-in search engine for your cluster objects using bleve.
